简介

Hexo 是一款基于 Node.js 的静态博客框架，它快速、简单且功能强大，可以很方便的生成静态网页并快速部署托管在 GitHub Pages 、Heroku、虚拟主机(FTP服务器)上面。

Maupassant

  • Github链接

  • 主题安装：

    1. 安装 Maupassant 主题和需要的渲染器

      ps:如果安装 npm install hexo-renderer-sass --save 报错，则需要进行以下处理：

      1. 安装 2.7 版本的 python

      2. 安装 淘宝 NPM 镜像

      3. 再安装 hexo-renderer-sass 渲染器即可(如果之前安装失败的话，需要卸载之前的 $ npm uninstall hexo-renderer-sass --save)

    2. 更换 _config.yml 配置文件的主题
    3. 最后 generate 并 deploy 即可
