Markdown概述

Markdown 是一种轻量级的「标记语言」，它用简洁的语法代替排版，而不像一般我们用的字处理软件 Word 或 Pages 有大量的排版、字体设置。它使我们专心于码字，用「标记」语法，来代替常见的排版格式。

Markdown语法汇总

文本

  • 加粗

    • 语法：**文本**
    • 示例：**放手一搏**
    • 效果：放手一搏

  • 斜体

    • 语法：*文本*
    • 示例：*放手一搏*
    • 效果：放手一搏

  • 加粗+斜体

    • 语法：***文本***
    • 示例：***放手一搏***
    • 效果：放手一搏

超链接

  • 文字链接

    • 普通方式：

      • 语法：[图片描述](链接)
      • 示例：[放手一搏](http://www.code1024.win")
      • 效果：放手一搏

    • 引用方式：

      • 语法：[图片描述][1]

      • 示例：

      • 效果：
        放手一搏

  • 图片链接

    • 普通方式：

      • 语法：![图片描述](链接)
      • 示例：![放手一搏](http://www.code1024.win/image/encouragement.jpg)

      • 效果：

    • 引用方式：

      • 语法：![图片描述][1]

      • 示例：

      • 效果：

  • 自动链接
    • 语法：<链接>
    • 示例：<http://www.code1024.win>
    • 效果：http://www.code1024.win

代码

  • 前后用 ` 包围起来。
  • 在前面加一个 Tab 缩进
  • 在前面加4个空格

引用

在需要引用的句子前面加上 > 符号

例如： >放手一搏

放手一搏

标题

  • 前面加上 # 号，注意 # 号后面需加上一个空格，#号个数代表标题级数。
  • 快捷键：Ctrl + 1，2，3，4(1对应一级标题，2对应二级标题…以此类推，共6级标题)

列表

  • 有序列表
    • 在前面加上 1. 2. 3. （点后面有一个空格）
  • 无序列表
    • 前面加上 - + * 等符号和一个空格即可

      分割线

      分割线的语法只需要另起一行，连续输入三个星号 * 即可。
